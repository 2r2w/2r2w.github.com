Performance Increase in Cycles

It’s really a easy to increas performance of your programm for about 10%: *just change cycles implementation * First I’ve seen this question on [stackoverflow][], then I’ve decided to check it out using g++ version 4.6.

about tests

test1.cpp (slow one)

#include <vector>
#include <stdlib.h>

int main(void) {
    std::vector<int> numbers;
    for (int i=0; i <1000000; ++i) 
        numbers.push_back(i);
    for (size_t i=0; i <numbers.size(); ++i) 
        numbers[i] += 10;
    return 0;
}

Compile:

g++ -o test1 test1.cpp

test2.cpp (fast one)

#include <vector>
#include <stdlib.h>
int main(void) {
    std::vector<int> numbers;
    for (int i=0; i <1000000; ++i) 
        numbers.push_back(i);
    for (size_t i=numbers.size(); i>0; --i) 
        numbers[i-1] += 10;
    return 0;
}

compilation process:

$ g++ -o test1o0 -O0 test-cycle1.cpp 
$ g++ -o test1o2 -O2 test-cycle1.cpp 
$ g++ -o test1o3 -O3 test-cycle1.cpp 
$ g++ -o test2o0 -O0 test-cycle2.cpp 
$ g++ -o test2o2 -O2 test-cycle2.cpp 
$ g++ -o test2o3 -O3 test-cycle2.cpp
